The Binary Blizzard Bot Utility Demo

About the Bot

This web application demonstrates the core **Deck.js** logic and key command features of the Binary Blizzards Bot, including card handling, mini-games, and a currency system.

ü§ñ Bot Command List

The bot supports the following global commands, as defined:

  • /test: Basic command.
  • /challenge: Challenge to Rock Paper Scissors.
  • /guess [suit] [value]: Guess a random card.
  • /rules: Show Blackjack rules (detailed below).
  • /balance: Show your current money balance.
  • /daily: Claim your daily free money.
  • /bj start [bet]: Start a Blackjack game.
  • /emoji: Get a random emoji.
  • /dadjoke: Get a random dad joke.
  • /tarot: Draw a single Tarot card for a reading. (NEW)
  • /wordle:Guess a 5 letter hidden word (NEW!)
